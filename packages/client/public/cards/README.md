# Card Art Assets

Place card PNG images in this folder. Each file should be named after the card's `id` field.

## Naming Convention

```
{card-id}.png
```

## Example

```
f-gray-fog-wisp.png
rp-ember-acolyte.png
n-street-urchin.png
```

## Specifications

- **Format**: PNG (transparent background recommended)
- **Size**: 200x300px (or any 2:3 ratio)
- **Style**: Portrait / character art, dark fantasy aesthetic matching Lord of the Mysteries theme
- **Opacity**: Images are rendered at 60% opacity with the pathway gradient behind them
